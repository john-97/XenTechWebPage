### Installation

#### Prerequisites: 
* **NodeJS (https://nodejs.org/en/download/)**

```
git clone https://github.com/john-97/XenTechWebPage.git

// RUN DEVELOPMENT SERVER (RUNS ON localhost:3000)
cd client
npm start

// RUN PRODUCTION BUILD (RUNS ON localhost:5000)
cd client
npm run build
serve -s build
```
